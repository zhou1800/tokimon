# AI Entry Point

Start sequence for work in the active Tokimon v2 repository:

1. Read this file.
2. Read `docs/helix.md`.
3. Read `docs/repository-guidelines.md`.
4. Load only the smallest set of docs and code needed for the current task.

Non-negotiables:

- User instruction is the top-level authority.
- Executable truth beats prose. Tests, benchmarks, schemas, and CLI behavior outrank docs when they conflict.
- Docs are an AI working memory layer for this repo, not a ceremonial artifact.
- If a doc is stale, update it in the same change rather than treating it as sacred.
- Use the root `docs/c4/` files for system context. `archive/` is historical only.
